# OMX Mail Client

**OMX Mail Client** is a terminal-based mail client written in Python. It features user authentication, mail management, spam handling, and account settings, all in a colorful command-line interface.

---

## Features

- Login / register with token-based authentication  
- Send mails with CC/BCC and preview  
- View Inbox, Sent, and Deleted (Trash) mails  
- Manage spam: add/remove senders  
- Search mails by subject, sender, or content  
- Change password, username, or delete account  
- Fully automatic dependency management and offline installation  
- Terminal animations and colored interface
